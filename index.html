Press Any Key to Start

Press Key here:

1. If You Are Playing In Mobile , then Press Enter Key in Mobile.(Zoom Out First).
2. Notice One Random Button Flashed With Sound then Tap On it.
3. Then Notice Second Random Button Flashed.
4. Then After Tap On First Flashed Button Then Second Flashed Button.
5. Then Again One Random Button Flashed.
Note : You Have To Remember The Sequences Of Buttons By Pressing It Along to Level Up.
If Pressing Button Sequence Is NOT Same As Flashing Button Sequence Then ' GAME OVER '.
